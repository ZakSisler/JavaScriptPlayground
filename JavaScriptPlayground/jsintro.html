Functions

Functions can be written in JS and called upon in reaction to an event. Allow the user to enter a color. When they click on a button (event), call a function that will display a message and the user's input.

Water Weight

Assume a gallon of water weighs 8.33lbs. Allow user to enter a number of gallons. Upon clicking a button, call a funtion to calculate the total weight and display that value.

Single Milestone

Allow user to enter their age. They click a button, a function is called that uses a switch that determines if they have milestones coming up.

Loops

Looping also exists in JS. Use a for loop to display 1-10.

Multiple Milestones

Allow user to enter their age. When they click a button, call a function that uses if/else to display all milestones they have reached.
