Introduction to JavaScript

JavaScript

  • JavaScript makes web pages more interactive
  • Reacts to events like clicks, keystrokes, page load, etc...
  • Combined with HTML and CSS we have a complete client-side solution to making a static HTML page feel dynamic.

jQuery

  • jQuery is a JavaScript Library originally built to use CSS selectors with an easier syntax
  • Utilize prebuilt JavaScript functions to manipulate objects on our HTML page
  • jQuery has a more concise syntax and allows us to ™Write Less, Do More

jQuery Plugins

  • jQuery plugins help to extend the jQuery functionality
  • Built to interact with a collection of elements
  • Give developers a large set of tools to make web pages more interactive
